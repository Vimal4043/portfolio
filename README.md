# Vimal Kumar Portfolio

Modern full-stack developer portfolio focused on clean UI, smooth interactions, and production-ready frontend engineering.

## Live Links

- Live Demo: https://portfolio-sage-nine-61.vercel.app/
- GitHub Repository: https://github.com/Vimal4043/portfolio

## Why This Portfolio

- Built with a mobile-first, responsive design approach
- Includes smooth motion transitions and micro-interactions
- Features dark/light theme support and sticky active navigation
- Highlights real-world projects with live demos and source code
- Includes working contact form integration for direct outreach

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion

## Featured Sections

- Hero, About, Skills, Projects, Achievements, Experience, Contact
- Project cards with GitHub and live demo links
- Scroll progress indicator, back-to-top, and polished footer

## Run Locally

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
npm run preview
```
